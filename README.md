Sales Forecasting Project (Time-Series Analysis)
This project focuses on predicting future sales using historical e-commerce data. The goal is to help businesses make data-driven decisions by forecasting demand, analyzing trends, and understanding seasonal patterns.

Project Overview

The objective of this project is to:
- Analyze historical sales data
- Identify trends, seasonality, and patterns
- Build forecasting models (ARIMA, Prophet)
- Visualize future sales predictions
- Create business insights using dashboards

Tools & Technologies Used
- Python
  - Pandas
  - NumPy
  - Matplotlib / Seaborn
  - ARIMA / SARIMA
  - Prophet
- Power BI (Dashboard)

Power BI Dashboard
The dashboard includes:
- Total Sales
- Total Profit
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Top Performing Products

Time-Series Models Used:

ARIMA Model
- Checked stationarity
- Differencing applied
- Auto ARIMA for parameter tuning
- Forecasted next 6 months of sales

Prophet Model
- Captured trend + seasonality
- Smooth curve for long-term prediction
- Confidence intervals included

Evaluation Metrics
- MAE
- RMSE
- MAPE

Key Insights:
- Clear seasonal spikes in specific months
- Overall sales increasing year-over-year
- Top products contribute to major revenue
- Forecast indicates continued growth
